# Wi-Fi Threat Detection System (AI-Powered)

## Overview
An AI-powered system that detects malicious Wi-Fi traffic in real time using machine learning.

## Features
- Live Wi-Fi traffic monitoring
- Random Forest based attack detection
- NSL-KDD style feature extraction
- Real-time dashboard with confidence score
- Firewall-style response capability

## Technologies Used
- Python
- Scapy
- Random Forest (Scikit-learn)
- Tkinter
- Google Colab (Model Training)

## How It Works
Packets are captured live, converted into NSL-KDD features, classified by a trained Random Forest model, and the AI’s decision is displayed in real time.
